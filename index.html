☁️ Connect Supabase — Free Cloud Sync

Sync HomeBase across your iPhone, iPad, and PC. Free forever on Supabase's hobby plan. Takes about 5 minutes.

Create your Supabase project
Go to supabase.com → Sign Up → New Project. Pick any name (e.g. "homebase"), choose a strong password, select US East region. Wait ~2 min for it to provision.
Enable Data API + Auto RLS
In your project: Project Settings → API → make sure "Enable Data API" is ON. Then Database → RLS → enable "Automatic RLS". Both should be on.
Create your tables + RLS policies
Go to SQL Editor → New Query → paste and run this entire block:
create table if not exists tasks (id text primary key, name text, cat text, pri text, date text, done boolean default false, who text, notes text, repeat text, completed_at text, created_at timestamptz default now()); create table if not exists maintenance (id text primary key, name text, date text, freq text, notes text, created_at timestamptz default now()); create table if not exists equipment (id text primary key, emoji text, name text, model text, serial text, warranty text, notes text, status text default 'ok', created_at timestamptz default now()); create table if not exists notes (id text primary key, title text, body text, created_at timestamptz default now()); create table if not exists projects (id text primary key, name text, cat text, budget int, spent int default 0, progress int default 0, start_date text, end_date text, description text, steps jsonb default '[]', bom jsonb default '[]', project_notes text, created_at timestamptz default now()); create table if not exists events (id text primary key, title text, date text, time text, end_time text, repeat text, color text, notes text, created_at timestamptz default now()); create table if not exists folders (id text primary key, grp text, name text, emoji text, sort_order int default 0); -- Allow full access (personal app — you control access via your secret key) do $$ declare t text; begin foreach t in array array['tasks','maintenance','equipment','notes','projects','events','folders'] loop execute format('alter table %I enable row level security', t); execute format('create policy if not exists "allow_all" on %I for all using (true) with check (true)', t); end loop; end $$;

✨ AI Assistant Setup

The AI needs an Anthropic API key. Because this runs in a browser, calls must go through a small proxy. Here's the free 2-minute setup:

Step 1 — Get your Anthropic API key
Go to console.anthropic.com/keys → Create Key → copy it.
New accounts get $5 free credit. HomeBase uses ~$0.01 per AI conversation.
Step 2 — Deploy a free Cloudflare Worker proxy (one time)
1. Go to workers.cloudflare.com → Sign up free → Create Worker
2. Replace all the code with this:
export default { async fetch(req) { if(req.method==='OPTIONS') return new Response(null,{headers:{'Access-Control-Allow-Origin':'*','Access-Control-Allow-Headers':'*','Access-Control-Allow-Methods':'POST'}}); const body = await req.json(); const key = req.headers.get('x-api-key'); const r = await fetch('https://api.anthropic.com/v1/messages',{ method:'POST', headers:{'Content-Type':'application/json','x-api-key':key,'anthropic-version':'2023-06-01'}, body:JSON.stringify(body) }); const data = await r.json(); return new Response(JSON.stringify(data),{headers:{'Content-Type':'application/json','Access-Control-Allow-Origin':'*'}}); } }
3. Click Deploy → copy your worker URL (e.g. my-proxy.username.workers.dev)
